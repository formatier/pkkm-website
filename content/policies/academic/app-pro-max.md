# แอพสภา Pro Max

นโยบายนี้ เป็นนโยบายส่งเสริมการใช้เทคโนโลยีสมัยใหม่อย่างเป็นระบบ โดยทางสภาจะเขียนเว็บไซต์โดยเขียนระบบขึ้นมาเอง ทำให้สามารถแก้ไขปัญหาต่าง ๆ ได้ โดยจะติดต่อขอข้อมูลนักเรียนจากกลุ่มบริหารวิชาการ เพื่อนำมายืนยันตัวตนนักเรียนที่เข้าใช้งาน อีกทั้งยังบูรณาการกับนโยบายส่วนมากของพรรคได้อีกด้วย

## จุดประสงค์

เพื่อจัดระบบต่าง ๆ ในโรงเรียนให้เป็นระบบ โดยการเขียนเว็บไซต์ และระบบหลังบ้านขึ้นมาเอง สามารถแก้ปัญหาต่าง ๆ ได้อย่างดีเยี่ยม เช่น ปัญหานักเรียนโกงคะแนนเสียง หรือการรับสมัครกิจกรรมต่าง ๆ

## รายละเอียด

ในการขอข้อมูลนักเรียน ทางสภาจะนำข้อมูลที่เป็นความลับมาเข้ารหัสด้วยกุญแจเฉพาะ ตัวอย่าง “KKW” เข้ารหัสเป็น “$2a$04$5UT” ซึ่งไม่สามารถถอดรหัสได้ และนำไปเก็บบันทึกในฐานข้อมูลได้อย่างปลอดภัย เมื่อนักเรียนป้อนข้อมูลเข้ามาจะนำข้อมูลนั้นมาเข้ารหัสด้วยกุญแจเดียวกัน ถ้าผลลัพท์ออกมาตรงตามข้อมูลที่บันทึกไว้ ก็จะอนุญาตให้เข้าใช้งาน ในส่วนของการออกแบบระบบหลักบ้าน ทางสภาจะตั้งเครื่อง Server 2 เครื่อง ถ้าเกิดเหตุ Server ล่ม อีก Server จะทำงานต่อได้อย่างปลอดภัย
